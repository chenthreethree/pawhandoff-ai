# PawHandoff AI

宠物代喂与寄养智能交接助手。它不负责“找人”，而是把主人零散的照护要求整理成标准交接包和每日任务，再将照护方的打卡、照片与备注汇总成给主人看的今日报告。

本仓库是用于早期机会验证的本地 Streamlit MVP。它不提供真实支付、定位监控、人员派单或安全担保，也不会把数据写入数据库。

## 本地运行

建议使用 Python 3.10 或更高版本。在当前目录执行：

```bash
pip install -r requirements.txt
streamlit run app.py
```

浏览器通常会自动打开 `http://localhost:8501`。

## MVP 功能

- 主人端表单：记录宠物习惯、喂养要求、空间边界、异常判断和紧急联系人。
- 一键生成：照顾说明书、每日任务、异常流程、照护方简版、出门准备清单及隐私提醒。
- 照护方打卡：完成项勾选、1–3 张照片上传、异常与补充备注。
- AI 今日报告：自动整理完成情况、宠物状态、饮食饮水排泄、异常、照片说明和微信简报。
- 二维码照护卡：展示本地概念二维码、宠物名与紧急联系人。
- Markdown 导出：下载交接包和今日报告。

## 演示路径

1. 进入“主人端生成交接包”，填写必填项并生成。
2. 切换到“代喂人 / 寄养方打卡”，勾选任务、上传图片并提交。
3. 在“今日报告”查看 AI 整理结果并下载 Markdown。
4. 在“二维码照护卡”查看概念卡片。

## 技术与数据说明

- UI：Streamlit
- 二维码：qrcode + Pillow
- 状态：仅使用 `st.session_state`，刷新/关闭会话后可能丢失
- AI：当前为规则驱动的 mock 生成；`ai_generate_handoff_pack()` 与 `ai_generate_daily_report()` 已预留稳定接口，可在后续替换为 OpenAI API 或其他大模型 API
- 照片：仅在当前 Streamlit 会话中用于计数和演示，不落库、不做识别
